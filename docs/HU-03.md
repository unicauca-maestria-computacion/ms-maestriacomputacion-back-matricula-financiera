# HU-03: Consultar periodos académicos disponibles

**As a** sistema frontend de gestión de maestría
**I want** obtener la lista de todos los periodos académicos registrados
**So that** pueda presentar al usuario las opciones de periodo para filtrar estudiantes matriculados

---

## Business Rules

- BR-01: Retorna todos los periodos registrados en la BD compartida, ordenados por fecha de inicio descendente.
- BR-02: Cada periodo incluye su estado (ACTIVO, INACTIVO, CERRADO).
- BR-03: Si no hay periodos registrados, retorna lista vacía (no es un error).

---

## Acceptance Criteria (BDD)

**Scenario 1 — Happy Path: BD con periodos registrados**
- Given: la tabla `periodo_academico` tiene 4 registros
- When: `GET /api/v1/gestion-matricula-financiera/periodos`
- Then: HTTP 200 con lista de 4 objetos `PeriodoAcademicoResponse` con id, tagPeriodo, fechas y estado

**Scenario 2 — Edge Case: BD sin periodos**
- Given: la tabla `periodo_academico` está vacía
- When: `GET /api/v1/gestion-matricula-financiera/periodos`
- Then: HTTP 200 con lista vacía `[]`

**Scenario 3 — Error: Fallo de conexión a BD**
- Given: la conexión a la base de datos no está disponible
- When: `GET /api/v1/gestion-matricula-financiera/periodos`
- Then: HTTP 500 con `ProblemDetail` donde `errorCode = "MF-0001"`

---

## Technical Specification

**Endpoint:** `GET /api/v1/gestion-matricula-financiera/periodos`
**Input port:** `ManageEnrolledStudentsUseCase.getAcademicPeriods()`
**Output port:** `StudentGatewayPort`
**Response DTO:** `List<PeriodoAcademicoResponse>`
**ErrorCodes:** MF-0001 (error genérico/BD)
**Security:** ROLE_MF_READ en prod, permitAll en dev
**BD tables:** `periodo_academico`
